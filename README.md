# AI Validation Copilot

## 📌 Overview

AI Validation Copilot is a prototype web application built for manufacturing industries.

The application helps validation engineers analyze bearing validation parameters and provides AI-based recommendations.

---

## 🚀 Features

- Validation data entry
- Historical validation comparison
- Risk Analysis
- AI Recommendation Engine
- Dashboard
- Decision Support

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
